## Idea 1 — "Dark Data Lab" (Neo-Brutalist Data Aesthetic)

**Design Movement:** Neo-Brutalism meets Data Visualization culture

**Core Principles:**
- Raw, unapologetic typography with heavy weight contrasts
- Data-inspired decorative elements (grid lines, dot matrices, terminal-style text)
- High contrast black/off-white with a single electric accent color
- Asymmetric layouts that feel like a well-organized data dashboard

**Color Philosophy:**
- Background: near-black `#0A0A0F`
- Surface: dark slate `#12121A`
- Accent: electric teal `#00E5CC`
- Text: crisp white `#F5F5F5`
- Muted: `#888` for secondary text
- Emotional intent: Precision, intelligence, technical mastery

**Layout Paradigm:**
- Sticky left rail with name + nav links (vertical)
- Right side scrollable content with generous padding
- Section headers use large monospace numbers (01, 02, 03...)
- Skills rendered as a tag cloud / grid matrix

**Signature Elements:**
- Animated terminal cursor blinking in the hero
- Horizontal rule lines styled as data pipeline connectors
- Code-block-style project cards

**Interaction Philosophy:**
- Hover reveals extra info in a tooltip-like overlay
- Smooth scroll with section snap
- Cursor changes to crosshair on interactive elements

**Animation:**
- Hero text types in character by character
- Cards slide in from the left on scroll
- Skill tags pop in with stagger delay

**Typography System:**
- Display: `Space Grotesk` (bold 700/900)
- Body: `JetBrains Mono` for technical feel
- Accent labels: `Space Grotesk` 500

## Idea 2 — "Warm Editorial" (Magazine-Inspired, Warm Neutrals) ✅ CHOSEN

**Design Movement:** Editorial / Architectural Digest meets Tech Portfolio

**Core Principles:**
- Warm, earthy tones that feel human and approachable despite technical content
- Generous whitespace with intentional typographic hierarchy
- Asymmetric two-column layouts with large section labels
- Data visualized as elegant infographic-style elements

**Color Philosophy:**
- Background: warm cream `#FAF8F5`
- Surface: white `#FFFFFF`
- Primary accent: deep forest green `#1A4A3A`
- Secondary accent: warm terracotta `#C4622D`
- Text: dark charcoal `#1C1C1E`
- Muted: `#8A8A8A`
- Emotional intent: Confidence, warmth, intellectual depth — standing out from the sea of blue tech portfolios

**Layout Paradigm:**
- Full-width hero with large name on the left, tagline on the right
- Sticky top nav with minimal links
- Alternating left/right content blocks for experience
- Projects in a 2-column masonry-style grid
- Skills organized in horizontal category rows

**Signature Elements:**
- Thin horizontal rule decorators with small diamond/dot motifs
- Large oversized section numbers in muted color behind section titles
- Subtle paper-texture background on hero section

**Interaction Philosophy:**
- Hover on project cards lifts them with warm shadow
- Nav links underline with a terracotta slide animation
- Smooth section transitions

**Animation:**
- Hero name fades in with slight upward drift
- Section elements animate in on scroll (fade + translate-y)
- Skill tags animate in with stagger

**Typography System:**
- Display: `Playfair Display` (bold, italic for emphasis)
- Body: `DM Sans` (clean, modern, readable)
- Mono: `JetBrains Mono` for code/tech labels

## Idea 3 — "Midnight Blueprint" (Dark + Geometric + Data-Viz Inspired)

**Design Movement:** Data Visualization meets Architectural Blueprint

**Core Principles:**
- Deep midnight blue as the primary surface — evokes depth and intelligence
- Geometric grid patterns as decorative motifs (blueprint lines)
- Accent colors pulled from data visualization palettes (amber, cyan)
- Clean sans-serif typography with strong weight hierarchy

**Color Philosophy:**
- Background: midnight navy `#0D1B2A`
- Surface: `#162032`
- Primary accent: golden amber `#F4A261`
- Secondary: cool cyan `#4CC9F0`
- Text: `#E8EDF2`
- Emotional intent: Depth, innovation, analytical rigor

**Layout Paradigm:**
- Full-width hero with blueprint-grid background texture
- Top navigation with logo/name on left
- Experience as a vertical timeline
- Projects in a 3-column card grid

**Signature Elements:**
- Blueprint-style dotted grid background on hero
- Glowing amber accent lines as section dividers
- Animated data flow lines connecting sections

**Interaction Philosophy:**
- Cards glow on hover with accent color border
- Smooth parallax on hero section
- Active nav item highlighted with amber underline

**Animation:**
- Hero section: particles/dots animate in
- Cards: fade-in with scale from 0.95
- Timeline items: draw in from left

**Typography System:**
- Display: `Syne` (geometric, distinctive)
- Body: `Outfit` (clean, modern)
- Mono: `Fira Code` for technical elements
